Explore geographics of users

On this page, we take a closer look at the geographical location of the users and trends regarding the different continents. Here we exclusively look at the users that actually have a location attached to them, which is only 949 out of the total 1585.

World Map Network

We explore the locations both using a map plainly showing the locations. Next an interactive map with nodes and edges. Feel free to click and explore.

It can be seen from the maps, that there are definitely some big hubs in North America defined by single very influential users. There is also a hub in Europe mainly around Paris and smaller hub in Asia around Beijing. There seems to be quite the connection across borders.
The number of connections across continents and within borders can be seen in the table below, where it is clear that most connections are made between the two main represented continents Europe and North America. Interestingly, there are more connections between these continents than within Europe itself showing the dominance of North American users on GitHub related to machine learning.

Overview

From the initial analysis of the continents, we again see an overall trend of using the Python programming language, however it seems that in Asia they are more into using JavaScript than in Europe where they prefer C#. In North America, these three languages are all represented in the top three most used. By inspecting the sentiments, no significant difference can be seen. Interestingly, it seems that the despite the less number of users from Oceania, the users on average have the second highest number of followers, that is 329.
